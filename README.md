# Employee Database Analysis using SQL and Tableau

## Project Overview

This project analyses an employee database using SQL and Tableau. The goal is to explore employee demographics, salary trends, and department-level insights through data analysis and visualisation.

## Tools Used

- SQL (MySQL)
- Tableau
- Excel/CSV

## Analysis Performed

The project includes analysis of:

- Male and female employees by department
- Male and female managers by department
- Average salary of male and female employees by department
- Average department salary compared with the company average

## SQL Skills Demonstrated

- Data extraction and filtering
- Joins
- Aggregate functions
- Common Table Expressions (CTEs)
- Window functions
- CASE statements
- Creating Procedures

## Tableau Dashboard

The Tableau dashboard provides visual insights into employee distribution, management structure, and salary comparisons across departments.

## Project Files

- **SQL/** - SQL queries used for analysis
- **Tableau/** - Tableau workbook and related files
- **Images/** - Dashboard screenshots

## Key Findings

The analysis helps identify differences in employee distribution, management representation, and salary patterns across departments.
